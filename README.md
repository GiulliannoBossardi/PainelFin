# FinPanel — Painel de Controle Financeiro

Aplicação web de gestão financeira pessoal, totalmente offline e sem servidor.  
Todos os dados ficam salvos no **localStorage** do navegador.

## 🚀 Acesso rápido

> **[Abrir o FinPanel →](https://SEU-USUARIO.github.io/NOME-DO-REPO/)**

## 📁 Estrutura de arquivos

```
├── index.html   → Estrutura da página (HTML)
├── estilo.css   → Estilos e temas (CSS)
├── script.js    → Lógica e dados (JavaScript)
└── README.md    → Este arquivo
```

## ✨ Funcionalidades

- 💰 **Salário** — registre adiantamento, pagamento e extras
- 📉 **Saídas** — despesas à vista ou parceladas, com status pago/pendente
- 📈 **Entradas** — receitas avulsas ou recorrentes
- 📊 **Controle** — resumo mensal de receitas, despesas e saldo
- 📦 **Investimentos** — controle de aportes e rendimentos
- ⚙️ **Configuração** — critérios, tipos de conta e usuários

## 💾 Sobre os dados

Os dados ficam salvos no **localStorage** do navegador. Limpar o cache apaga tudo.  
Faça backups periódicos em **Configuração → Backup**.
